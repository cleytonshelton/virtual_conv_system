What I’m Doing

I’m building a fully virtual conveyor system capstone that integrates PLC logic, HMI monitoring, operator controls, and 3D emulation into one synchronized workflow.

  • Developing and validating conveyor control behavior with PLC logic.
  • Using Emulate3D to model conveyor motion, sensors, and product flow.
  • Using FactoryTalk View and Ignition for virtual controls and live system visibility.
  • Documenting testing, integration status, and troubleshooting results.

Why I’m Doing It

Real conveyor systems are expensive and risky to test against during early development. This virtual approach enables safer, faster, and lower-cost validation before physical deployment.

  • Reduce training and prototyping risk by testing in a controlled virtual environment.
  • Improve reliability by catching integration and communication issues earlier.
  • Support operator training with realistic system behavior and real-time dashboards.
  • Create a repeatable process for future automation projects.

Project Outcome

The goal is a complete virtual conveyor demonstration that proves end-to-end control, monitoring, and synchronization across all connected tools.
